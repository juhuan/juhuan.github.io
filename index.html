Mac 给照片添加 GPS 信息

偶然在微信看到一篇相机获取 GPS 定位信息的文章，想到之前单反甚至黑卡都没有 GPS 信息甚是不方便，趁假期有空就想实践下。基本思路：

  1. 相机和手机同步时间
  2. 手机记录 GPS 信息
  3. 读取 gpx 写入照片 exif

2. 手机记录 GPS 信息

原文中的手机软件还要收费，想到六只脚貌似也有导出 gpx 的功能，就直接用六只脚吧，手机里现成的软件。

记录完之后将运动的 gpx 导出，传输到 Mac 端。

3. 读取 gpx 写入照片 exif

本想按部就班直接用 lightroom 的，但是。。我 Mac 上的 lightroom 过期了，只能退而求其次，寻找替代品。APP Store 搜了下都是收费软件，PASS。去 github 碰碰运气，还真的找到一个不错的软件！GeoTag
